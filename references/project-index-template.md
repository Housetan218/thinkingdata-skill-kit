## Field Notes

- `Project ID`: main project ID used inside ThinkingData
- `Aliases`: user-facing names or common shorthand
- `Subprojects Confirmed`: `yes`, `partial`, or `no`
- `Dossier Status`: `missing`, `draft`, `verified`
- `SQL Template Status`: `missing`, `partial`, `verified`
- `Last Verified`: absolute date, not relative wording

## Maintenance Rule

Whenever a project dossier or query template is updated, also update this index.
